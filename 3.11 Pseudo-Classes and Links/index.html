Example for Pseudo-Classes with links

Anchor tag with 'href' attribute

  • When a link is unvisited, its color is lightgreen
  • When a link is visited, its color is rosybrown

Anchor tag without 'href' attribute

  • When this link is idle, it is in darkgreen color
  • When hovering over this link, it changes to yellowgreen color
  • When clicking on this link, it changes to mediumvioletred color
  • When focusing this link with TAB key, it changes to goldenrod color
